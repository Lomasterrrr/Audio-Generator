# AudioGenerator
Generates a sound given: volume, frequency, duration!

![alt text](https://i.imgur.com/tw7Eeqt.png)

## Start
1. Download build.zip
2. Unpack zip
3. Run exe

## Options
1. Enter: **volume** (recommended: 0,2)
2. Enter: **frequncy**
3. Enter: **duration**
4. Enter: **audio wave** (recommended: 1)
5. **Done!**
